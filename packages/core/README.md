# @emartai/mindr-core

Internal shared library for [Mindr](https://github.com/emartai/mindr). Contains the git watcher, tree-sitter convention detector, memory storage backends (SQLite + Remembr), session context builder, bug fingerprinter, debt detector, and quality scorer.

**This package is the internal engine. Most developers should install [`mindr`](https://www.npmjs.com/package/mindr) (CLI) or [`@emartai/mindr`](https://www.npmjs.com/package/@emartai/mindr) (SDK) instead.**
